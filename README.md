# Data Analyst Portfolio

A modern and responsive Data Analyst portfolio built with React and Vite. The portfolio showcases my projects, skills, experience, and contact information with a clean UI and interactive design.

## Features

* Responsive design
* Modern UI and animations
* Projects showcase
* Skills section
* Contact section
* Fast performance with Vite

## Tech Stack

* React
* Vite
* JavaScript
* HTML & CSS

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
